### Installing

**Cloning the Repository**

```
$ git clone https://github.com/willcordeiro/ExpenseControlReact

$ cd ./<project page>
```

**Installing dependencies**

```
$ yarn
```

_or_

```
$ npm install
```

**Starting the project**

```
$ npm start
```

the project will run at the default react port http://localhost:3000/
